# sisfokol_yayasan

SISFOKOL YAYASAN v1.0 , khusus untuk kalangan internal yayasan, agar bisa memantau sekolah - sekolah yang dimiliki.

Dibuat dan dites dengan WebServer XAMPP PHP 7.4 . Pada LinuxMint.




---


FITUR :

- Multiple Sekolah

- Kepegawaian Sekolah

- Keuangan Sekolah

- Sarpras Sekolah

- Forum

- FileBox Dokumen

- Inbox Pesan
